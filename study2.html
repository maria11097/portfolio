대체 텍스트

alt로 대체 텍스트 제공

  • 텍스트가 있는 이미지를 이미지 요소로 마크업할 때 대체 텍스트는 이미지 요소의 alternative속성인 'alt'에 적용하면 된다.
  • img 요소뿐만 아니라 button, 이미지맵의 area에도 대체 텍스트가 존재한다면 alt로 제공.

<img src='../img/img1.jpg' alt='올 여름 개발자의 완벽한 계획 부스터와 함께하기'>

마크업으로 대체 텍스트 제공

  • 접근성에 필요하지만 화면에는 구현을 안 했으면 하는 요소가 있다.
  • 디자인에 맞추기 위해 text-indent, display:none, visibility:hidden을 사용할 때가 있는데 이 요소들은 리더기가 읽어주지 않기 때문에 다른 방법을 사용 해야 한다.(개인적으로 clip:rect(0 0 0 0)을 사용)
<p class="blind">blind</p>

데이터 혹은 복잡한 컨텐츠

  • 데이터 혹은 복잡한 컨텐츠는 내용이 많기 때문에 테이블로 대체 텍스트를 제공하고 있다.
  • 데이터가 있는 차트는 보통 테이블로 많이 사용한다.

longdesc

  • img 요소에 대체 텍스트가 길어지면 별도로 제공하는 longdesc 값을 넣어준다.
  • longdesc는 img 속성 중에 하나로 이미지 대체 텍스트 길이가 길어 사용한다. 하지만 유지보수가 힘들다.

의미 없는 이미지

  • 의미 없는 이미지라고 해서 alt="" 속성을 적어주지 않으면 리더기가 파일명을 읽기 때문에 alt="" 속성을 적어주되, 빈 공간으로 놓는다. 
<p>오늘만 무료 배송!</p>

QR Code

  • 시각적으로 인지 할 수 없기 때문에 alt="qr code"라고 하지 말고 링크 주소를 적어준다. 
<img src="../images/qrcode.png" alt="http://www.naver.com 바로 가기 QR코드">

  • 문구가 적혀져 있는 썸네일과 설명글이 있을 경우 리더기에서 2번 읽어주기 때문에 사용자가 혼동을 겪을 수 있다. 이럴 경우, alt="" 속성을 빈 값으로 놓는다.

보안 문자 이미지
